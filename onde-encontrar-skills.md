## 8. CHECKLIST DE AVALIAÇÃO DE SKILLS

Quando encontrar um repositório, avalie:

```
✅ QUALIDADE
  [ ] Mais de 100 stars? (indica uso real)
  [ ] Últimas atualizações recentes? (< 3 meses)
  [ ] README claro e completo?
  [ ] Exemplos funcionais fornecidos?

✅ COMPATIBILIDADE
  [ ] Funciona com Claude ou pode adaptar?
  [ ] Dependências estão claras?
  [ ] Documentação menciona a versão do modelo?

✅ REUTILIZAÇÃO
  [ ] Código modular? (não um monólito)
  [ ] Licença permite uso comercial?
  [ ] Customizável para seus casos de uso?

✅ PERFORMANCE
  [ ] Token usage documentado?
  [ ] Latência aceitável mencionada?
  [ ] Custo estimado claro?
```

## 10. WORKFLOW RECOMENDADO

### 1️⃣ **Discovery (30 min)**
```bash
# Busque no GitHub
github.com/search?q=seu_caso_use language:python stars:>50

# Confira PromptBase
promptbase.com → search por tipo

# Veja docs oficiais
docs.anthropic.com → "Prompt Library"
```

### 2️⃣ **Avaliação (15 min)**
```
✓ Qualidade (stars, recência, docs)
✓ Compatibilidade (modelo, dependências)
✓ Reutilização (modularidade, licença)
```

### 3️⃣ **Adaptação (1-2 horas)**
```
1. Clone o repositório
2. Copie o prompt/código relevante
3. Adapte pra seu contexto (modelos, dados)
4. Teste com dados reais
5. Integre com suas skills
```

### 4️⃣ **Documentação**
```
Adicione à sua skill:
- Fonte original (link + author)
- Adaptações feitas
- Casos de uso específicos
- Performance metrics
```
