🚀 COMEÇAR EM 5 MINUTOS
═════════════════════════════════════════════════════════════════════════════

1️⃣ Clone seu repositório (se não tiver)
   git clone git@github.com:adrianotomasoni/skills.git
   cd skills

2️⃣ Copie os scripts para seu repositório
   mkdir -p scripts/
   cp sync-all.sh sync-to-claude.sh sync-to-cursor.sh scripts/
   chmod +x scripts/*.sh

3️⃣ Rode sincronização completa
   ./scripts/sync-all.sh

4️⃣ Pronto! ✅
   • Claude acesso em: /mnt/skills/user/
   • Cursor carregou .cursorrules
   • GitHub foi atualizado

═════════════════════════════════════════════════════════════════════════════
🔄 3 FORMAS DE SINCRONIZAR
═════════════════════════════════════════════════════════════════════════════

OPÇÃO 1: TUDO DE UMA VEZ (Recomendado)
────────────────────────────────────────
   ./scripts/sync-all.sh
   
   ✓ Sincroniza Claude
   ✓ Sincroniza Cursor
   ✓ Valida Git
   ✓ Oferece opção de push

   Tempo: 2-3 minutos
   Frequência: Sempre que adicionar/editar skills


OPÇÃO 2: INDIVIDUAL (Mais Controle)
────────────────────────────────────
   ./scripts/sync-to-claude.sh
   ./scripts/sync-to-cursor.sh
   
   ✓ Sincroniza um ambiente por vez
   ✓ Útil pra testes
   
   Tempo: 30 segundos cada


OPÇÃO 3: MANUAL (Último Recurso)
────────────────────────────────
   # Claude
   cp skills/**/SKILL.md /mnt/skills/user/[skill-name]/
   
   # Cursor
   cp .cursorrules ~/projects/[project]/.cursorrules
   
   # GitHub
   git add skills/
   git commit -m "Update skills"
   git push origin main
   
   Tempo: 5 minutos
   Quando usar: Se os scripts não funcionarem

═════════════════════════════════════════════════════════════════════════════
⚙️ SETUP AVANÇADO (Opcional)
═════════════════════════════════════════════════════════════════════════════

Adicione alias ao bash/zsh:

  # ~/.bashrc ou ~/.zshrc
  alias sync-skills='~/skills/scripts/sync-all.sh'
  
Depois:
  sync-skills    # Executa tudo!

Ou configure cron (atualização automática a cada hora):

  0 * * * * cd ~/skills && ./scripts/sync-all.sh >> ~/sync.log 2>&1

🚨 TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

"Permission denied" ao rodar script
  chmod +x scripts/*.sh

"Não encontra /mnt/skills/user"
  mkdir -p /mnt/skills/user
  ./scripts/sync-all.sh

"Git conflicts"
  git pull origin main
  Resolva conflicts manualmente
  git commit -m "Resolve conflicts"

"Claude não vê skills novas"
  1. Rode sync-all.sh
  2. Regenere contexto em Claude.ai (nova conversa)
