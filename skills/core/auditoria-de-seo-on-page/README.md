# Auditoria de SEO On-Page

Skill de referência para auditar o SEO on-page de **uma página avulsa ou externa**
(qualquer site/cliente que não seja o `traderisk.com.br`).

Avalia title, meta description, headings, conteúdo/keywords, imagens/alt, links
internos, dados estruturados (schema) e performance básica, produzindo um
**score 0–100** e um **plano de ação priorizado**.

Para auditar o domínio `traderisk.com.br` e suas páginas estratégicas, use a skill
`seo-audit-traderisk`.

Ver `SKILL.md` para o checklist completo, a tabela de pesos e os erros comuns.
