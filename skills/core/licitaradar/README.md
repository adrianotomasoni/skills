# LicitaRadar

Skill para monitoramento de licitações públicas via PNCP com classificação de oportunidades e detecção de exigências de seguro garantia.

## Versão

**1.0.0** (Beta)

## Estrutura

```
licitaradar/
├── SKILL.md              # Definição da skill
├── README.md             # Este arquivo
├── api-integration/
│   ├── pncp-client.ts    # Cliente TypeScript para PNCP
│   └── claude-classifier.ts  # Classificador via Claude
└── examples/
```
