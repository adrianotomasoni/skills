# SEO Audit TradeRisk

> Categoria: `core` · Auditoria e otimização de SEO do site `traderisk.com.br`.

## O que é

Skill especializada em SEO para o nicho de seguro garantia, crédito comercial e fintechs B2B brasileiras. Audita e otimiza o domínio `traderisk.com.br` e suas páginas estratégicas (ex.: `/seguro-garantia`, `/seguro-credito`), o blog e o conteúdo institucional, cobrindo on-page, conteúdo (E-E-A-T), análise técnica (Core Web Vitals, mobile, schema) e palavras-chave do nicho. Gera relatório com score e recomendações priorizadas.

## Gatilhos (quando acionar)

- "Audita o SEO da página /seguro-garantia"
- "Otimiza o SEO do blog da TradeRisk"
- "Revisa o SEO do site traderisk.com.br"
- "Quais keywords priorizar para o nicho de seguro garantia?"

## Quando NÃO usar

- Página avulsa ou externa que NÃO seja `traderisk.com.br` → `auditoria-de-seo-on-page`.

## Como usar por plataforma

- **Claude Code / Codex / Gemini / Copilot:** instale em `~/.claude/skills/seo-audit-traderisk/`.
- **Claude.ai:** upload de `SKILL.md` + `benchmarks.json` e informe a página a auditar.
- **Cursor:** `@skills/core/seo-audit-traderisk/SKILL.md` + URL/conteúdo da página TradeRisk.

## Exemplo

Prompt: *"Audita o SEO de /seguro-garantia."*
Saída esperada: relatório com score geral, pontos fortes, tabela de oportunidades por prioridade, keywords sugeridas e próximos passos. Ver `examples/relatorio-exemplo.md`.

## Material de apoio

- `SKILL.md` — metodologia, keywords prioritárias e formato de saída.
- `benchmarks.json` — benchmarks do setor (Core Web Vitals, taxa de rejeição, tempo na página, competidores).
- `examples/relatorio-exemplo.md` — relatório preenchido (genérico).
