# Template — Artigo de blog (SEO)

> Para conteúdo editorial do traderisk.com.br. 1200–2000 palavras. Otimizar para a keyword-alvo sem keyword stuffing. Antes de publicar, rodar `seo-audit-traderisk`.

## Estrutura

```
H1: [keyword principal + benefício] (50–60 chars no title)
  - Intro (gancho + promessa do que o leitor vai aprender) — 2–3 parágrafos
H2: [Subtema 1 — a dor / contexto]
H2: [Subtema 2 — como funciona / solução]
  H3: [detalhe]
  H3: [detalhe]
H2: [Subtema 3 — quando faz sentido / exemplos]
H2: Perguntas frequentes (bom para featured snippet)
  H3: [Pergunta 1]
  H3: [Pergunta 2]
Conclusão + CTA único
```

## Checklist on-page

- [ ] Title 50–60 chars com keyword no início
- [ ] Meta description 120–160 chars com keyword + CTA
- [ ] Um único H1 com a keyword
- [ ] Keyword e variações semânticas de forma natural
- [ ] Links internos para páginas relacionadas (ex.: /seguro-garantia)
- [ ] CTA ao final
- [ ] Imagens com alt descritivo

## Exemplo de esqueleto (genérico)

```
H1: Seguro garantia judicial: quando ele substitui o depósito recursal
Intro: gancho sobre travamento de caixa em depósitos + promessa de guia prático.
H2: O que é o seguro garantia judicial
H2: Depósito recursal x seguro garantia
  H3: Impacto no caixa
  H3: Aceitação pelos tribunais
H2: Como contratar e em quanto tempo sai a apólice
H2: Perguntas frequentes
CTA: "Fale com um especialista TradeRisk."
```
