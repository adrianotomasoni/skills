# Codex

**Formato:** mantém o frontmatter.

## Instalar
Instale em `~/.codex/skills/<id>/` ou no alias `~/.agents/skills/<id>/`
(o Codex reconhece o alias cross-runtime).
```bash
cp -r skills/<categoria>/<id> ~/.agents/skills/<id>
```

## Usar
- Skills carregam nativamente; siga as instruções apresentadas quando a skill ativa.

## Notas
- Mantém frontmatter; sem strip.
- Instalar uma vez em `~/.agents/skills/` cobre Codex + Copilot + Gemini CLI.
