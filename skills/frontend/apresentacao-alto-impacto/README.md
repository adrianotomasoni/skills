# Apresentação de Alto Impacto

Criação e revisão de apresentações corporativas e de produto com acabamento premium (nível
consultoria/branding) para a TradeRisk — pitches, decks comerciais e materiais para
investidor/cliente. Foco em narrativa, dados visuais e CTA forte.

## Quando usa (gatilhos)

- "Monta um deck de / cria uma apresentação de produto"
- "Preciso de slides para reunião / cliente / investidor"
- "Transforma este material institucional em apresentação"
- **Não usar para:** texto corrido (use `docx`) ou slides puramente analíticos sem narrativa.

## Como usar por plataforma

- **Claude Code / Cowork:** siga o método passo a passo do `SKILL.md` (briefing → narrativa → storyboard → design → revisão → ensaio).
- **Montagem do deck:** use `templates/estrutura-deck.md` como esqueleto e valide cada slide com `templates/slide-checklist.md`.

## Exemplo

> "Transforme este bullet em um slide."
> Bullet: "Reduzimos custos, melhoramos eficiência e aumentamos visibilidade."
> → a skill converte em um slide de **uma ideia**, com título-frase forte, um benefício
> principal e um visual de apoio (os demais viram slides próprios).

## Material de apoio

- [`SKILL.md`](SKILL.md) — método, exemplos e common mistakes.
- [`templates/estrutura-deck.md`](templates/estrutura-deck.md) — estrutura Capa → Problema → Solução → Como funciona → Prova → CTA.
- [`templates/slide-checklist.md`](templates/slide-checklist.md) — checklist de qualidade por slide.
