# Checklist de Qualidade por Slide

Passe por este checklist em **cada** slide antes de considerar o deck pronto. Se um item
falhar, corrija ou corte o slide.

## Uma ideia

- [ ] O slide comunica uma única ideia central.
- [ ] É possível resumir o slide em uma frase (o título-frase).
- [ ] Conteúdo que pertence a outra ideia foi movido para outro slide ou cortado.

## Hierarquia

- [ ] O elemento mais importante é o mais proeminente (tamanho/peso/posição).
- [ ] O título é uma frase com a mensagem, não um rótulo genérico ("Solução").
- [ ] O olho cai primeiro no ponto certo; apoio é claramente secundário.

## Contraste e legibilidade

- [ ] Texto legível à distância / em tela compartilhada (fontes não minúsculas).
- [ ] Contraste suficiente entre texto e fundo.
- [ ] Máximo de ~6 linhas de texto; sem parágrafos corridos.
- [ ] Significado não depende só de cor (reforçar com ícone/rótulo).

## Dados visuais

- [ ] Dados apresentados como visual quando possível (gráfico > tabela > texto).
- [ ] O número/insight que importa está destacado.
- [ ] Gráficos têm título, unidade e fonte; sem poluição (chartjunk).
- [ ] Toda afirmação tem prova (número, cliente, demonstração).

## Consistência de marca

- [ ] Paleta, fontes e logo seguem a identidade. TODO: padrões de marca TradeRisk
- [ ] Estilo de gráficos, ícones e espaçamento consistente entre slides.
- [ ] Alinhamento e margens uniformes (grid).

## CTA

- [ ] O slide final tem um CTA específico e de baixa fricção.
- [ ] Há um único próximo passo claro (não várias opções concorrentes).
- [ ] Canal de contato/ação presente. TODO: contato oficial TradeRisk

## Narrativa (deck como um todo)

- [ ] Lendo só os títulos em sequência, o argumento se entende sozinho.
- [ ] Cada slide avança a história; nenhum é "preenchimento".
- [ ] O deck respeita o arco: problema → solução → prova → ação.
